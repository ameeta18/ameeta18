### 🔭 Featured Projects

#### 🛰️ [Multimodal Deep Learning for Landslide Susceptibility](https://github.com/ameeta18/Landslide-Prediction-Project)
A gated fusion architecture that predicts landslide risk from satellite data, designed to stay reliable when optical sensors fail during rain or cloud cover.

- **Precision 0.97 / Recall 0.83** on a geographically separated test set
- Significantly outperformed Random Forest baseline (0.79 / 0.49)
- Network autonomously learned the underlying physics: terrain gate 94.6%, moisture gate 74%
- Zero spatial data leakage by design

`PyTorch` `Rasterio` `GeoPandas` `NASA earthaccess` `Scikit-learn`

#### 🤖 [AI Research Assistant — Agentic RAG Pipeline](https://github.com/ameeta18/AI-Research-Assistant)
End-to-end RAG system that automates academic research through agentic LLM orchestration.

- **87% reduction in LLM context usage** through chunking, embedding, and retrieval
- **5× more papers per session** vs. naive context-stuffing
- LangGraph ReAct agent orchestrating 5 custom LangChain tools (arXiv search, PDF reader, vector indexer, semantic search, LaTeX renderer)
- Custom evaluation framework for retrieval accuracy, faithfulness, and chunk coverage

`LangGraph` `LangChain` `FAISS` `Google Gemini` `Streamlit` `Python`
